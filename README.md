PROJECT OVERVIEW
------------------------------------------------------------------------
Savi Dabba is a modern, responsive web application for a South Indian-
inspired cloud kitchen. The project features a dynamic menu, a shopping 
cart system, and user authentication, all wrapped in a premium 
"Dark & Orange" glassmorphism design.

"Savi Dabba" captures the authentic, heartwarming taste of a home-cooked 
meal in the bustling heart of Bangalore.

------------------------------------------------------------------------
KEY FEATURES
------------------------------------------------------------------------
1. Responsive Design
   - Fully optimized for Mobile, Tablet, and Desktop screens.
   - Built using Bootstrap 5 grid system and custom media queries.

2. Dynamic Menu Categories
   - Breakfast, Lunch, Dinner, Desserts, and Beverages.
   - Special "Featured" layouts for high-priority items.

3. Shopping Cart System
   - Add/Remove items dynamically.
   - Persists data using LocalStorage (cart is saved even if you refresh).
   - Real-time total calculation.

4. User Authentication
   - Login and Registration functionality.
   - Powered by Google Firebase Authentication.

5. Modern UI/UX
   - "Fire & Ice" Theme: Orange (#ff6600) & Black (#000000).
   - Glassmorphism: Translucent cards with background blur effects.
   - Smooth hover animations and interactive carousels.

------------------------------------------------------------------------
TECHNOLOGIES USED

* Framework:    Bootstrap 5.3.2
* Backend:      Google Firebase (Auth & Realtime Database)

------------------------------------------------------------------------
SETUP & INSTALLATION
------------------------------------------------------------------------
1. Download or Clone the project folder.
2. Ensure you have an internet connection (for Bootstrap/FontAwesome CDNs).
3. Open 'index.html' in any modern web browser (Chrome, Edge, Firefox).
4. Note: To use the Login/Register features, ensure the Firebase config 
   in 'cart.js' is active and valid.
